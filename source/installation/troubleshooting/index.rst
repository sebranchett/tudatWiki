Troubleshooting
===============

If you encounter any problems during the installation of Tudat, this section of the wiki most likely provides information to get your Tudat installation up and running. It contains a list of well known errors encountered in the past. Furthermore, some troubleshooting tools are described which help to pinpoint any problems.
