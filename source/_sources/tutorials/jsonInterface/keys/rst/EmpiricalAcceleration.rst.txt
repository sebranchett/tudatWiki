
.. role:: jsontype
.. role:: jsonkey
.. role:: arrow

- :jsontype:`number` :jsonkey:`constantAcceleration` (optional). Acceleration (in RSW frame) that is constant. Default value: :literal:`[0, 0, 0]`.
- :jsontype:`number` :jsonkey:`sineAcceleration` (optional). Acceleration (in RSW frame) that scales with sine of true anomaly. Default value: :literal:`[0, 0, 0]`.
- :jsontype:`number` :jsonkey:`cosineAcceleration` (optional). Acceleration (in RSW frame) that scales with cosine of true anomaly. Default value: :literal:`[0, 0, 0]`.
